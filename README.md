A simple Minecraft clone made in Python in 3 days or more using [Ursina Engine!](https://www.ursinaengine.org/)\
This project is in 	<ins>no way affiliated</ins> with [Mojang or Minecraft](https://www.minecraft.net/).\
Developed by [weet-ops](https://github.com/weet-ops).

# Features
Around 31 blocks to use!\
Easy customization! Really!\
And much more!

# How To Play Minceraft
Run the <sup>"download-all-dependencies"</sup> file (if you don't already have ursina) and then run the "minceraft.py" file and enjoy!

# Why was Minceraft created?
Boredom.

# Controls (as of version 1.2-alpha)
>J - Next block\
>K - Previous block\
>WASD -  Move\
>Space - Jump\
>TAB - Save (added in v1.2-alpha)\
>G - Load (added in v1.2-alpha)\
